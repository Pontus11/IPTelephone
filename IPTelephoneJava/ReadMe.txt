Simple ip telephone with a server.

Main class for the ip telephone client is PhoneGuiFX.java
Main class for the server is IPTServer.java

Uses SQLite as an embedded database and can do everthing ranging from creating a new user, handle loggins, handeling chat histories,
place and receive IP calls, add friends and more. 

The project was created as part of a solo university project with a very short deadline of only a few days.
